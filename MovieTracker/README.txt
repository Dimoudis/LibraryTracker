Library Tracker - Python Project

Περιγραφή:
Πρόγραμμα για την προσθήκη και την προβολή βιβλίων ή ταινιών σε μια βιβλιοθήκη.
Τα δεδομένα αποθηκεύονται σε JSON αρχείο.

Οδηγίες Χρήσης:
1. Τρέξε το αρχείο tracker.py
2. Επιλέξτε:
   1 - Προσθήκη βιβλίου ή ταινίας
      1.1 - Πληκτρολόγησε τίτλο, κατηγορία και βαθμολογία (1-10).
   2 - Προβολή βιβλιοθήκης
   3 - Έξοδος

Αρχεία:
- tracker.py : Κύριο πρόγραμμα
- library.json : Αρχείο δεδομένων

# Library Tracker - Update 1

# Τι νέο υπάρχει
- Προσθήκη **αμυντικού προγραμματισμού**:
  - Ο τίτλος δεν μπορεί να είναι κενός.
  - Η κατηγορία δέχεται μόνο 'book' ή 'movie' (ανεξαρτήτως πεζών/κεφαλαίων).
  - Η βαθμολογία πρέπει να είναι αριθμός από 1 έως 10.
  - Αποφυγή προσθήκης στοιχείου με ήδη υπάρχον τίτλο.

- Προσθήκη διαγραφής στοιχείων από τη βιβλιοθήκη.
